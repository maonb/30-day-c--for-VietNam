### 🏦 Project: Hệ thống Ngân hàng đơn giản
**Tính năng:**
- Đăng ký / Đăng nhập tài khoản
- Gửi tiền, rút tiền, chuyển tiền
- Xem lịch sử giao dịch
- Lưu dữ liệu vào file

**Kỹ thuật sử dụng:**
- OOP: Class, Inheritance, Polymorphism
- STL: vector, map
- File I/O, Exception handling
- Smart pointers, Namespace
- Multi-file + Makefile
