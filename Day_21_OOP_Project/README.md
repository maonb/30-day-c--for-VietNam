# Day 21: Bài tập tổng hợp Tuần 3 — Mini Project OOP

## 🎯 Mục tiêu
- Áp dụng tất cả kiến thức OOP từ tuần 3
- Xây dựng 1 mini project hoàn chỉnh

## 📝 Chọn 1 trong các project sau:

### Project A: Quản lý Sinh viên
- Class `Student` với tên, MSSV, điểm các môn
- Class `Classroom` quản lý danh sách sinh viên
- Tính điểm trung bình, xếp loại
- Tìm kiếm, sắp xếp theo tên/điểm
- Xuất danh sách ra console

### Project B: Quản lý Thư viện
- Class `Book`, `Member`, `Library`
- Mượn/trả sách, kiểm tra tình trạng
- Tìm kiếm sách theo tên/tác giả
- Hiển thị lịch sử mượn

### Project C: Hệ thống Hình học
- Abstract class `Shape`
- Các class con: `Circle`, `Rectangle`, `Triangle`
- Tính diện tích, chu vi
- So sánh hình (operator overloading)
- Quản lý collection of shapes

## 💡 Yêu cầu chung
- Sử dụng inheritance & polymorphism
- Có menu tương tác console
- Code clean, có comment
