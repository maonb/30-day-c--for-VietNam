# Day 14: Bài tập tổng hợp Tuần 2

## 🎯 Mục tiêu
- Ôn tập kiến thức tuần 2 (Day 08–13)
- Thực hành hàm, con trỏ, tham chiếu, bộ nhớ động

## ✏️ Bài tập

### Bài 1: Linked List đơn giản
Tạo danh sách liên kết đơn với các thao tác: thêm, xóa, tìm kiếm, in.

### Bài 2: Quản lý mảng động
Viết chương trình quản lý mảng số nguyên động (thêm, xóa, resize).

### Bài 3: String processing
Viết các hàm xử lý chuỗi dùng con trỏ (strlen, strcpy, strcat tự viết).

### Bài 4: Callback pattern
Dùng con trỏ hàm để tạo hàm `filter()` lọc mảng theo điều kiện.

### Bài 5: Matrix calculator
Viết chương trình tính toán ma trận (cộng, nhân) dùng bộ nhớ động.
